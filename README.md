# KickStart

## Features

- [x] Show events from API endpoint
- [x] Read events from a database
- [x] Authentication
- [x] Sign up for an event
- [x] List users signed up for an event
- [ ] Events backoffice/CRUD
- [ ] Leave a comment on an event
